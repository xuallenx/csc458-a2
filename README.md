1. Overview

This assignment investigates the phenomenon of bufferbloat using Mininet.


Topology:
bw-host: bandwidth of h1 → s0 (default: 1000 Mb/s)

bw-net: bandwidth of s0 → h2 (required argument)

delay: link delay on both links (ms)

maxq: queue size (in packets) on bottleneck link

time: duration of experiment


How to run:

Simply CD into scripts folder and run sudo ./run.sh

sudo ./clean.sh is used to clean
